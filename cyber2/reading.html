1. What is a Port?

A port is a numbered “doorway” on a device that network traffic uses to reach a specific service. Ports range from 0–65535.

Example: SSH often uses port 22, and HTTP often uses port 80.

2. What is SSH?

SSH (Secure Shell) is a protocol used to securely access and manage computers remotely. Attackers often target SSH because it can provide direct access to a system if credentials are guessed or stolen.

3. Why Repeated Attempts Matter

When you see many IPs repeatedly attempting to connect to a service (like SSH), it can indicate scanning or brute-force behavior.

Even if no login succeeded yet, repeated attempts increase risk.

4. Firewall Restriction

A firewall rule can block or restrict inbound access to sensitive ports. A common best practice is to allow SSH only from trusted networks (like your school or VPN).
